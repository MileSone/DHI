refreshToken(successCallback, errorCallback)

For OAuth, the SDK can not refresh because it does not persist client credentials.

For OAuth, the SDK can not refresh because it does not persist client credentials. the SDK can not persist credentials from the users, so it can not successfully do a refresh. This function only exists here because it inherits from the Authorization object, which is also used for other types of authentication in which the token can expire.
