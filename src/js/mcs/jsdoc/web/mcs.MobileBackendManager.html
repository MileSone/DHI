Class: MobileBackendManager

mcs. MobileBackendManager

new MobileBackendManager()

The entry-point into the Mobile Cloud Service SDK.

The entry-point into the Mobile Cloud Service SDK. The MobileBackendManager has a singleton from which MobileBackend objects can be accessed, which in turn provide access to Analytics, Storage, Auth and other capabilities. The singleton can be accessed as mcs.MobileBackendManager.
