Class used to authorize a user against Facebook and use the OAuth token from Facebook to authenicate against the Mobile Cloud Service.

Class used to authorize a user against Facebook and use the OAuth token from Facebook to authenicate against the Mobile Cloud Service. Callers should use MobileBackend's FacebookAuthorization() property.

authenticate(successCallback, errorCallback)

Authenticates a user with the given credentials against Facebook.

Authenticates a user with the given credentials against Facebook. The user remains logged in until logout() is called. In the Facebook Developer console the user must define the domain where the user application will access from. When the user adds a platform for the application, the user will chose Website and set the site URL to http://localhost/ in the Facebook Developer UI.
